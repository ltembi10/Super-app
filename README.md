# Campus Life Super App

## Project Description
The Campus Life Super App is a web application designed to help students stay connected with campus activities. It provides information about events, displays dynamic media using an external API, and offers an interactive user experience.

## Purpose
The goal of this project is to demonstrate full-stack web development skills using:
- Frontend technologies (HTML, CSS, JavaScript, Bootstrap)
- Backend development (Flask)
- Database management (SQLite)
- API integration

---

## Technologies Used

### Frontend:
- HTML5 (semantic structure)
- CSS3 (custom styling + media queries)
- Bootstrap 5 (responsive design)
- JavaScript (event-driven interactivity)

### Backend:
- Python (Flask framework)
- SQLite (database)

### Other:
- External API (Giphy API)
- BeautifulSoup (web scraping)
- Requests (API calls)
